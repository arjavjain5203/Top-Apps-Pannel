# Top Apps Pannel

✨ *This is a vibe-coded project.* ✨

Top Apps Pannel is a sleek and simple taskbar extension for GNOME Shell that enhances your desktop workflow. It integrates your favorite and currently running applications directly into the GNOME top panel, giving you quick access to your most-used tools without needing to open the overview.

## Features

- **Integrated App Icons:** Displays both your favorite applications and currently active windows directly on the top panel.
- **Smart Auto-Hide:** When a window is fully maximized, the top panel intelligently slides away to give your application 100% of the screen space.
- **Hover Reveal:** While the panel is hidden, simply move your mouse to the top edge of the screen to smoothly slide the panel back into view. The panel will stay visible as long as you are interacting with it, and slide away once you leave.
- **Multi-Window Indicators:** Clear visual indicators show which apps are running and easily distinguish apps that have multiple windows open.
- **Highly Customizable:** 
  - Adjust panel position (Left, Center, Right)
  - Control icon sizes, padding, and desaturation
  - Toggle notification badges and progress bars directly on the icons
  - Configure multi-monitor behavior and workspace isolation

## Installation

### From GNOME Extensions (Recommended)
You can install Top Apps Pannel directly from the [Official GNOME Extensions](https://extensions.gnome.org/) website once it is published.

### Install From Source
To install directly from the source code:
1. Clone this repository to your local extensions directory:
   ```bash
   git clone https://github.com/arjavjain5203/Top-Apps-Pannel.git ~/.local/share/gnome-shell/extensions/topappspro@arjav/
   ```
2. Restart GNOME Shell (Press `Alt+F2`, type `r`, and press `Enter` on X11, or log out and log back in on Wayland).
3. Enable the extension via the **Extensions** app or **GNOME Tweaks**.
